## 🎯 Purpose

This site serves as my digital presence — a place to highlight my projects, share my background, and provide a way to get in touch.

## ✨ Features

- Responsive design (works on desktop, tablet, and mobile)
- Clean and minimal UI
- Project showcase section
- About / Bio section
- Contact / social links

## 🛠️ Technologies

HTML5 — Structure

CSS3 — Styling and layout

JavaScript — Interactivity and dynamic content

## 🔧 Customization

The personalisation of this portfolio for your own use is not allowed without due credits.

-- Replace images in the /img folder

-- Update text content in index.html

-- Modify colours and fonts in /css

-- Adjust interactive behaviour in script.js
