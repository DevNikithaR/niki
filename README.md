## 🎯 Purpose

This site serves as my digital presence — a place to highlight my projects, share my background, and provide a way to get in touch.

## ✨ Features

- Responsive design (works on desktop, tablet, and mobile)
- Clean and minimal UI
- Project showcase section
- About / Bio section
- Contact / social links

-- Update text content in index.html
